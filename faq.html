FAQ

AVENIR is a joint initiative of EUROCONTROL and EASA that aims to support aviation environmental sustainability by developing indicators, tools, methodologies, and best practices to monitor and improve environmental performance of aviation, including CO₂ and non-CO₂ climate impacts.

The catalogue provides an overview of tools, dashboards, methodologies, and use cases that can support ANSPs in monitoring, analysing, and improving environmental performance.

The catalogue is primarily intended for: Air Navigation Service Providers (ANSPs).

AVENIR does not necessarily develop new operational tools. It identifies, documents, and assesses existing tools, methodologies, and best practices, and helps define environmental indicators and frameworks that can be used across the aviation system.

Different tools are designed for different purposes (e.g. flight analysis, network monitoring, scenario modelling, reporting). No single tool can answer all types of questions, so the catalogue helps users identify which tool is appropriate for which type of analysis.

Results may differ because tools:

  • Use different methodologies
  • Use different input data
  • Cover different system boundaries
  • Work at different levels (flight, airport, network, scenario)
  • Are designed for different purposes (operational analysis, monitoring, reporting, forecasting)

This does not mean one tool is correct and another is wrong - they are designed for different decision-making contexts.

In general, results from different tools should not be directly compared without understanding the methodology, assumptions, and scope of each tool. 

Users should first define their question, for example:

  • Operational improvement analysis
  • Environmental performance monitoring
  • Scenario or policy assessment
  • Environmental reporting

Then select the tool that is designed for that purpose.

For detailed calculation principles and commonly used factors, users can consult:

  • EUROCONTROL Standard Inputs
  • Tool-specific methodology documentation (linked in each tool page)

Open‑Source Governance: The AVENIR Catalogue is developed as an open‑source, collaborative reference. Its evolution is driven by operational experience, user feedback, and contributions from ANSPs, ensuring that methods, tools and practices remain aligned with real-world challenges and emerging needs.
